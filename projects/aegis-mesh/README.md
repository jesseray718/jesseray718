# AEGIS MESH
The decentralized network protocol.
Ungovernable. No central server. No admin key.

## Network Layer
- Discovery: BLE + Wi-Fi Direct + Sneakernet (USB/SD)
- Sync: CRDT — works completely offline
- Validation: Proof of Physical Work reputation system
- Intelligence: Local LLM on Android (Termux + llama.cpp)

## Node Classes
Seed → phone + scripts running
Sprout → 1m² panel poured and curing
Tree → full thermal loop operational
Grove → food systems integrated
Forest → regional mesh, multiple nodes

## Mesh Properties
- No single point of failure
- Works without internet
- Self-healing — if one node goes down mesh reroutes
- Ungovernable by any single entity
- Grows by replication not by permission

## Protocol Stack
Layer 1: Physical node (aerocement, Stirling, food)
Layer 2: Mesh network (BLE, WiFi, sneakernet)
Layer 3: Intelligence (local LLM, sensor logging, AI)
