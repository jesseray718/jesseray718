# ACRE Token — Proof of Physical Work

> **Status: Not yet launched.** ACRE tokens do not currently exist.
> Launch requires: 3 verified nodes in different climates + legal opinion
> on utility token classification. No financial promises are made here.
> This document describes the intended design only.

## What ACRE Will Be

A utility token minted only when verified physical work is performed.
No pre-mine. No speculation. Supply grows only when the physical world grows.

## What Will Earn ACRE

| Activity | Tokens | Verification |
|----------|--------|--------------|
| Verified node build | 500 | Sensor data + photos + 2 validators |
| Bounty won | Per bounty | Prototype + blueprints + replication |
| Food produced per kg | 1 | Logged + spot check |
| Skills contribution | 50 | Peer review + merge |
| Research published | 200 | Validation + IPFS hash |
| Builder onboarded | 25 | Verified first build |
| Validator service | 500 | Per validation completed |

## What ACRE Will Buy
- Land time-shares on OpenRoot demonstration sites
- Resource credits (cold hours, energy, processing time)
- Governance votes on project direction

## Governance Design
Quadratic voting. No entity holds more than 2% voting power.
Founder holds no special keys. No veto. Community supreme.
