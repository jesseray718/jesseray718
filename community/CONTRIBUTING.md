# Contributing to OpenRoot
This project belongs to everyone.

## Highest Impact: Build a Node
Read projects/aerocement/docs/START-HERE.md
Pour a panel. Log data. Submit build log. You are now a node.

## Compete for a Bounty
See bounties/README.md. Build it. Validate it.
Blueprint enters the commons forever.

## Contribute to Skills Library
Know how to do something useful?
Document it in skills-library/ with step-by-step instructions.

## Become a Validator
Physically inspect builds near you.
Open Issue: "Validator Application — [Your Location]"

## Hard Rules
1. No claiming >100% thermodynamic efficiency
2. 21-day cure is non-negotiable — never soften this
3. Labyrinth is FILLED SOLID — never say lined or walled
4. Two separate tanks — never one combined tank
5. Desiccant at fresh air intake only
6. No patents on anything touching this commons
7. Failures are data — document honestly
8. Serve the least first

*The project succeeds when it no longer needs Jesse McMillen.*
