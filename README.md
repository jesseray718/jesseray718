What OpenRoot Is
Three systems, working together:
Immutable open hardware library — Every design is CC-BY-SA 4.0 forever. No patents. Ever.
Open knowledge commons — Community-built, peer-reviewed skills and research anyone can use, improve, and build on.
Real-value currency (ACRE) — Tokens minted only for verified physical work. No speculation. No inflation by decree.
The Thermal Loop
The first validated OpenRoot technology. A passive energy system that moves more useful energy than it consumes — not by violating thermodynamics, but by harvesting ambient heat differentials across a carefully staged loop.
